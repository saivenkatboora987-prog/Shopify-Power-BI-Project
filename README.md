1.Dashboard Title & Headline
Shopify Analysis:Shopify Sales and Customer Funnel Report
Shopify, an e-commerce platform that allows individuals and businesses to create online stores handling physical locations and process payments.
A dynamic and interactive Data Visualization Report to analyse Shopify transaction performance and Customer behaviour.
2.Purpose
The Shopify dashboard focuses on transaction performance and customer behaviour through analysing Business metrics(KPI's) like transaction performance, Customer Purchase Behaviour and Retention and value.
3.Technology Used
The Shopify Dashboard was build using these tools
 Power BI->This tool helpful for creating dashboards.
 Power Query->Transforms and clean the messy data into without bugs data.
 DAX Expression(Data Analysis Expression)->Basic Aggregation Functions, Logic and Filter Functions, Airthmetic Operations are used.
 Data Modelling->Data Modelling is not required in this dashboard, because the entire data because the entire analysis is built using a single table file.
 File Format ->.pbix for development and .png for previews.
 4.Data Source
 It's a dummy data of containing data for one week.The dataset consists of 7,431 rows and 19 columns.The data was focused on USA Country.The Data contains Order number, Customer ID, City,Province, Zip Code, Invoice date, Gateway, Product ID,Quantity,Total price USD and TAX.
5.Business problem
  Small companies using shopify often lack the tools to move beyond basic tracking to understand the overall health sales operations and customer interactions.
6.Goal of the Dashboard
  The goal of this dashboard is uncover meaningful insights,make decisions to enhance the future growth of using this shopify.
7.Walkthrough of Key Visuals
  KPI Cards,field and bubble maps, bar charts(area/bar charts),donut and column charts.
